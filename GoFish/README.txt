# Welcome to Anissa's and Pragna's Go Fish Game
------------------------------------------------

#Introduction
-- this code simulates the classic game of Go Fish
-- no input is needed 
-- there are no Jokers in this game
-- the winner will be the one with more cards in the book

#Requirements
-- two players only both must named Anissa and Pragna

#Installation
-- to compile type make
-- to run the file type ./fish
-- make sure you are running it on the most updated version of Linux

#FAQ
-- you should have none
